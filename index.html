Periodic Table Info

WARNING: This project is still in development and is not complete yet! Expected release date is the end of September 2016.

Run this program from the Python Module (IDLE) and you will see a list of all the elements on the Periodic Table display on the screen. Type the name of the one you would like to learn more about on the input line and all the information you'll ever want to know displays for easy viewing! This app was developed for education. To be a reference to, and also teach people about The Periodic Table. It has now branched out to become a piece of software for anybody who want to learn about the Periodic Table or have a reference to the Periodic Table on their computer that they can use anytime!

Features

  • Displays all of the elements in the Periodic Table
  • Displays even more detailed (but easy to read) information once you choose one.
  • Displays all this information from top sources like Wikipedia, and Basher.
  • Displays the information in a clean and easy to read interface. Making it easy for children.

Purpose

This project was originally developed for use in education, to make school aged children Periodic Table Wiz Kids. It has since been modified to be used by anyone who need a quick and easy reference to the Periodic Table and its Elements and wants to become a Periodic Table Wiz.

  • Download the Python Module if you haven't already: Download Link,and install it,
  • Extract the file from you GitHub Download,
  • Open the main.py file in the Python Module (IDLE),
  • And the app will open for you to use!

Use Instructions

After opening the app, read through the list of Periodic Table Elements and find the one you want to learn more about.

Type the name of that element at the bottom on the input line where it says: "Please enter the name of the element you would like to learn more about:". It does not matter how you type it in, as long is their is no punctuation, because the app automatically converts the input to lowercase.

After typing the name in successfully (the app will let you know if you don't), the app will display more information about the element you are trying to research.

Some of the information that it will display is as follows:

  • The name of the element,
  • A small, but detailed description of the element,
  • The elements number,
  • Its symbol,
  • Atomic number,
  • Atomic weight,
  • Color,
  • and much more!

After learning about one element you can search again to learn about many of the elements or maybe even become a Periodic Table Element wiz!
